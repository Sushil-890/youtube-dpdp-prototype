# YouTube Privacy & Consent Redesign (DPDP Act 2023)

A Product Management case study and interactive React prototype demonstrating how YouTube could redesign its privacy and consent experience to comply with India's Digital Personal Data Protection (DPDP) Act, 2023 while maintaining a seamless user experience.

## Features

- Privacy Center
- Granular Consent Management
- Data Download & Deletion
- Child Account Protection
- Product Manager Analytics Dashboard
- Responsive Material Design UI
- Built with React + TypeScript + Tailwind CSS

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- React Router

> **Note:** This is a Product Management prototype created for the Vedantu Product Management Internship assignment. It is intended to demonstrate UX, product thinking, and privacy-first design, not production-ready backend functionality.
